# Weather App

A simple weather web application built with HTML, CSS, JavaScript and Rust.

## Project Goal

The goal of this project is to allow users to search for a city and display its current weather information.


## Features

### Current Features

* Search for a city.
* Display the current weather.
* Show
  * City name
  * Temperature
  * Weather condition
  * Humidity
  * Wind speed
* Display an error message if the city is not found.

## Planned Features

* User geolocation
* Favorite cities
* 5-day weather forecast
* Dark mode
* Search history
* Multiple languages

## Technologies

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Rust
